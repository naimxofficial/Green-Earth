# 🌿 Green Earth

> **Plant a Tree, Grow a Future.**

Green Earth is a global tree plantation campaign website where users can browse trees by category, view detailed info, add trees to a cart, and donate to the cause. Built to raise awareness and make contributing to a greener planet as easy as a few clicks.

## ✨ Features

| Feature | Details |
|--------|---------|
| 🌳 **Tree Gallery** | Browse all available trees fetched live from an API |
| 🗂️ **Category Filter** | Filter trees by category with dynamic buttons loaded from API |
| 🔍 **Tree Detail Modal** | Click any tree image to open a detailed popup with description, category & price |
| 🛒 **Add to Cart** | Add any tree to your cart with one click |
| 🔢 **Quantity Tracking** | Adding the same tree multiple times increments its quantity |
| ❌ **Remove from Cart** | Remove individual items from the cart instantly |
| 💰 **Live Total Price** | Cart total updates in real-time as items are added or removed |
| 📭 **Empty Cart Message** | Friendly message shown when the cart has no items |
| ⏳ **Loading Spinner** | Smooth loading indicator while trees are being fetched |
| 🌍 **Our Impact Section** | Highlights 500K+ trees planted, 120+ communities & 30+ countries |
| 📋 **About the Campaign** | Overview of Green Earth's mission and environmental goals |
| 🌱 **Plant a Tree Form** | Donation form with name, email & number of trees input |
| 📱 **Fully Responsive** | Optimized for mobile, tablet, and desktop screens |
| 🎨 **Clean UI/UX** | Nature-themed green design with smooth section navigation |
